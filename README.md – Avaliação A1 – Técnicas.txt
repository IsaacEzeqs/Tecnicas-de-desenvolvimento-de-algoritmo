 Sobre o Repositório

Este repositório contém os quatro exercícios desenvolvidos para a Avaliação A1, seguindo exatamente as orientações descritas na tabela da atividade.

Cada exercício está separado em um arquivo .py, organizado da seguinte forma:

 avaliacao-a1-python
│
├── condicionais.py
├── repeticao.py
├── listas.py
├── dicionarios.py
└── README.md

 Descrição dos Exercícios
 1. condicionais.py

Programa que simula a verificação de idade para entrada em eventos.
Envolve uso de if, elif e else.

Funcionalidades:

Entrada da idade

Validação e condições

Exibição da permissão de entrada


 2. repeticao.py

Exibe números pares de 1 a 100 usando:

Estrutura for

Estrutura while

Inclui lógica de operadores e controle de loop.

 3. listas.py

Recebe nomes de alunos e armazena em uma lista.
Ao final, exibe todos os nomes cadastrados.

Inclui:

Entrada dinâmica

Uso de append()

Loop para impressão

4. dicionarios.py

Cadastro simples de produtos, onde o usuário informa:

Nome do produto

Preço

Os dados ficam armazenados em um dicionário Python.

Exibe todos os produtos cadastrados ao final.
